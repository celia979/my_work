# Mini_printf

## Description
Reproduction de la fonction printf en C avec gestion des différents formats.

## Fonctionnalités
- Gestion des formats standards (%d, %s, %c, %x…)  
- Parsing des arguments  
- Affichage formaté sur la sortie standard  

## Technologies
- C  
- Compilateur GCC  

## Comment utiliser
1. Cloner le repo : `git clone <lien>`  
2. Aller dans le dossier Mini_printf  
3. Compiler : `gcc -Wall *.c -o mini_printf`  
4. Lancer : `./mini_printf "format string" [arguments]`
