# Projet_graphic

## Description
Projets graphiques développés avec des bibliothèques graphiques (My Hunter et My Radar).

## Fonctionnalités
- My Hunter : jeu type “shooting” avec cibles et score  
- My Radar : simulation radar avec affichage des objets en mouvement  

## Technologies
- C  
- Bibliothèque graphique (SFML / SDL / autre selon projet)
