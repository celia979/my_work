# My_radar

## Description
Simulation d’un radar avec détection d’objets et affichage dynamique.

## Fonctionnalités
- Simulation de positions d’objets  
- Calculs mathématiques pour la trajectoire  
- Affichage graphique simple  

## Technologies
- C  
- Graphiques (bibliothèque utilisée selon le projet)  

## Comment utiliser
1. Cloner le repo : `git clone <lien>`  
2. Aller dans le dossier My_radar  
3. Compiler : `gcc *.c -o my_radar -l<library>`  
4. Lancer : `./my_radar`
