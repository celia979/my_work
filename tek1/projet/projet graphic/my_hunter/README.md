# My Hunter

## Description
Jeu graphique de type “shooting” développé en C, où le joueur doit toucher des cibles pour marquer des points.

## Fonctionnalités
- Cibles mobiles avec score  
- Gestion des événements clavier et souris  
- Boucle principale du jeu avec rafraîchissement graphique  

## Technologies
- C  
- Bibliothèque graphique utilisée (SFML / SDL / autre selon projet)  

## Comment lancer
1. Cloner le repo : `git clone <lien>`  
2. Aller dans le dossier My_Hunter  
3. Compiler : `make` ou `gcc *.c -o my_hunter -l<library>`  
4. Lancer : `./my_hunter`
