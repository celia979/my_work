# Minishell2

## Description
Implémentation d’un shell Unix en C, capable de lancer des commandes, gérer les processus et les redirections.

## Fonctionnalités
- Exécution de commandes simples et avec arguments  
- Gestion des redirections `>`, `<`  
- Gestion des pipes `|`  
- Commandes internes comme `cd`, `exit`  

## Technologies
- C  
- Linux / Terminal  

## Comment utiliser
1. Cloner le repo : `git clone <lien>`  
2. Aller dans le dossier Minishell2  
3. Compiler : `make` ou `gcc *.c -o minishell`  
4. Lancer : `./mysh`
