# Amazed

## Description
Projet de résolution de labyrinthes utilisant des algorithmes de recherche.

## Fonctionnalités
- Lecture d’un labyrinthe depuis un fichier  
- Calcul du chemin optimal  
- Affichage du labyrinthe avec solution  

## Technologies
- C  
- Algorithmes (DFS, BFS)  

## Comment utiliser
1. Cloner le repo : `git clone <lien>`  
2. Aller dans le dossier Amazed  
3. Compiler : `gcc *.c -o amazed`  
4. Lancer : `./amazed maze.txt`
